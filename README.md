# databases
Projetos em SQL

Habilidades adquiridas


*Sistema de E-commerce*

- Modelagem de clientes PF e PJ com restrição de exclusividade
- Estrutura de pedidos com múltiplas formas de pagamento
- Controle de entrega com status e código de rastreio
- Relacionamento entre clientes, pedidos, produtos e pagamentos
- Organização de tabelas com foco em escalabilidade
- Aplicação de boas práticas de modelagem

*Sistema de Universidade*

- Modelagem de alunos, professores, cursos e disciplinas
- Relacionamento entre alunos e disciplinas (matrícula)
- Controle de turmas, horários e professores responsáveis
- Estruturação de dados acadêmicos (notas e frequência)
- Implementação de relacionamentos N:N com tabelas intermediárias
- Aplicação de normalização para evitar redundância
- Garantia de integridade referencial entre entidades
- Organização do banco para consultas eficientes

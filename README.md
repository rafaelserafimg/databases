*Sistema de Companhia*

- Sistema de Companhia
- Criação de banco de dados no MySQL
- Modelagem de tabelas com chaves primárias e estrangeiras
- Relacionamento entre tabelas (1:N e N:N)
- Criação de consultas SQL com JOIN, GROUP BY e ORDER BY
- Uso de índices para melhorar desempenho das consultas

- Controle de acesso com usuários e permissões (GRANT)
- Implementação de procedures para manipulação de dados

- Organização de dados pensando em desempenho e integridade

- Criação de atributos derivados em consultas (cálculos com SUM)

- Criação de procedures com tratamento de erro
- Manipulação de dados com INSERT para testes
- Realização de backup e recovery com mysqldump
